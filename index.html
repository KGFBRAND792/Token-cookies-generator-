Facebook Login

Login karke Access Token paayein

Token Types:
- EAAD Token: Cookie login se milta hai (Events Manager/Ads)
- EAAA Token: Email/Password se milta hai (Mobile Token)

Authenticator app se 6-digit code ya Facebook 2FA setup ki secret key

EAAD Token ke liye Cookie Method use karein!

Cookies kaise lein?
1. Facebook.com open karein aur login karein
2. F12 press karein (DevTools)
3. Network tab mein jaayein
4. Koi bhi request select karein
5. Headers section mein Cookie copy karein

Warning: Apna Access Token aur Cookies kisi ke saath share mat karein. Ye sensitive information hai jo aapke account ko access kar sakti hai.
